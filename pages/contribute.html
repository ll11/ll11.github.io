This website (does.hcq.work) lives on Github.com.

It is built with Github Page+Jekyll+Documentation theme.

Fork this repo, then if you use Linux, just do "bundle update" and "bundle exec jekyll serve"

For Windows or Mac, follow the documentation.

The homepage is generated from index.md, create pull-request to help improving the homepage.

All studies are in _post/ folder, if you want to add a new article, follow the filename and content format. After you test it, you can start a pull-request.

If you have a technical problem, create an issue. Don't use Github Issues for political debate, but feel free to create an issue if you find grave error in the site's content.
